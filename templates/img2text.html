API Documentation

Use the HTTP API to integrate OCR into your own applications.

Endpoint Method Content-Type Description
/api/ocr POST multipart/form-data Extract text from an uploaded image
Request Parameters
Field Type Required Description
file File Yes Image file (PNG, JPG, JPEG, BMP, TIFF, WebP)
model String No OCR engine — easyocr (default) or tesseract 
curl -X POST http://localhost:5000/api/ocr \
  -F "file=@image.png" \
  -F "model=easyocr"
import requests

url = "http://localhost:5000/api/ocr"
files = {"file": open("image.png", "rb")}
data  = {"model": "tesseract"}

resp = requests.post(url, files=files, data=data)
print(resp.json())
